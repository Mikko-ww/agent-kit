# Examples

## Start a Task

1. Create `task_plan.md`, `findings.md`, and `progress.md`.
2. Fill in the goal and phases.
3. Mark Phase 1 as `in_progress`.

## During Work

- Research finding: update `findings.md`.
- Phase complete: update `task_plan.md` and `progress.md`.
- Error occurs: log it in both `task_plan.md` and `progress.md`.
